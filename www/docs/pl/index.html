Dokumentacja urirun

urirun to CLI dla pakietów komend adresowanych przez URI. Projekt deklaruje URI raz i wywołuje je z shella, backendu, frontendu, przepływu usług albo projekcji narzędzi dla agenta.

  • Nazewnictwo - co używa urirun i dlaczego URL repozytorium na GitHubie nadal zawiera urihandler.
  • Komendy - komendy CLI i wersjonowane punkty wejścia.
