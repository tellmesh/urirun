# Logo

The generated logo family lives in `logo/`:

- `urirun-icon.svg`
- `urirun-icon-mono.svg`
- `urirun-wordmark.svg`
- `urirun-horizontal.svg`
- `urirun-stacked.svg`
- `urirun-favicon.svg`
- `urirun-logo-sheet.svg`

The mark combines a URI colon and double slash with a compact motion shape for
the "run" part of the name.

The PHP site in `www/` uses copied assets under `www/assets/` so it can be
served directly from the `www` document root.
