# GitHub-only integration

## JavaScript
```bash
npm install github:tellmesh/urihandler
```

## Python
```bash
pip install "git+https://github.com/tellmesh/urihandler.git@main#subdirectory=adapters/python"
```

## Firmware / C
Copy files from `adapters/c/` into the firmware project.

## Recommended repo URL
```txt
https://github.com/tellmesh/urihandler
```
